01. Fonts

Note 1. Desktop font sizes should be in vw units, based on 1920px viewport witdh.

To calculate vw font size from a given pixel size, use the following formula

font-size: calc({font-size-in-px} / 1920 * 100vw);

For example, if you want to apply font-size 154px for desktop (width 1928), font size will be:

font-size: calc({font-size-in-px} / 1920 * 100vw);

output:

font-size: 8.0208333333vw
